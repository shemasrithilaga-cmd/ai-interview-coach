# 🎯 AI Interview Coach

An interactive **AI-powered Interview Preparation Platform** built using **Streamlit** that helps students practice technical and HR interviews, analyze their performance, and improve their skills with personalized feedback.

---

## 🚀 Features

### 🧠 Interview Practice

* HR, Technical (Python, DBMS, OS, Data Structures) questions
* Step-by-step mock interview flow
* Timer-based answering system
* Hint support for difficult questions

### 📊 Performance Analysis

* Real-time scoring system
* Feedback based on:

  * Keywords
  * Clarity
  * Answer structure
* Visual analytics using charts (Plotly)
* Session history tracking

### 🧪 Skill Assessment

* Multiple-choice skill tests
* Covers:

  * Python
  * Data Structures
  * DBMS
  * Operating Systems
  * System Design
* Instant results with explanations

### 🐞 Debugging Practice

* Real-world coding/debugging questions
* Company-based problems (TCS, Infosys, Wipro, etc.)
* Identify bugs and explain fixes

### 🤖 AI Mentor (Chatbot)

* ChatGPT-like mentor system
* Ask anything related to:

  * Interview preparation
  * Concepts explanation
  * Career guidance
* Provides smart, contextual responses

### 📚 Learning Roadmaps

* Beginner → Intermediate → Advanced paths
* Structured preparation guidance
* Topic-wise learning progression

### 💻 Practice Platforms Integration

Quick access to:

* LeetCode
* HackerRank
* Codeforces
* CodeChef
* GeeksforGeeks

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/ai-interview-coach.git
cd ai-interview-coach
```

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run the app

```
streamlit run app.py
```
